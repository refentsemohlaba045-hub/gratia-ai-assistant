# Gratia AI Assistant

A comprehensive AI assistant with voice capabilities, available for desktop, mobile, and web platforms.

## Features

- **Voice Input & Output**: Natural speech recognition and text-to-speech
- **User Authentication**: Secure login system with password hashing
- **Themes**: Space and Ocean themes with customizable emojis
- **Learning System**: AI that learns from conversations and improves over time
- **Cross-Platform**: Desktop GUI, mobile app (works offline), and web interface
