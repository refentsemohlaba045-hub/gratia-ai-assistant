Quick Start

1. Install Dependencies

cd c:\Users\Computer
uv pip install anthropic openai speech_recognition pyttsx3 sounddevice numpy scipy tkinter

2. Set API Keys (Optional)

For online AI responses, set your API keys:

$env:ANTHROPIC_API_KEY = "your-key-here"
$env:OPENAI_API_KEY = "your-key-here"

3. Run Your Chosen Version

# CLI Version
uv run python "c:\Users\Computer\OneDrive\import os.py"

# Desktop GUI
uv run python "c:\Users\Computer\OneDrive\gratia_gui.py"

# Mobile (Desktop Test)
uv pip install kivy
uv run python "c:\Users\Computer\OneDrive\gratia_mobile.py"

4. Create a Test User

In any version, type:

register refentse puv57MQN

Then login with: login refentse puv57MQN
