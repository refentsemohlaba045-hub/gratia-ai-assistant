# Gratia Mobile for Android

This is the Gratia AI assistant packaged for Android using the Kivy Launcher.

## Installation Instructions

1. Install the Kivy Launcher app on your Android device from the Google Play Store or download the APK from:
   https://github.com/kivy/kivy-launcher/releases

2. Copy this entire `GratiaMobile` folder to your Android device's SD card in the following location:
   `/sdcard/kivy/GratiaMobile/`

3. Open the Kivy Launcher app on your Android device

4. Select "GratiaMobile" from the list of available apps

5. Tap "Launch" to start the Gratia AI assistant

## Features

- Offline AI responses
- Voice input/output support
- Second brain learning system
- Ghost mode for background operation
- Sync with laptop brain (when online)
- Jarvis-style interface

## Requirements

- Android device with Kivy Launcher installed
- Microphone permissions for voice input
- Internet connection for online AI features (optional)

## Troubleshooting

If the app doesn't appear in Kivy Launcher:
- Make sure all files are in `/sdcard/kivy/GratiaMobile/`
- Check that `android.txt` and `main.py` are present
- Restart the Kivy Launcher app

For voice features to work:
- Grant microphone permissions to Kivy Launcher
- Speak clearly when using voice input
